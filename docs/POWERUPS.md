# پاورآپ‌ها (کارت‌های سوپرپاور) — پیاده‌سازی (فاز ۲)

> کد: `backend/src/powerups/`. اثرها **سرور-محور**اند (تایمر/درستی سمت سرور).

## ۱) چرا فقط تک‌نفره؟

قانونِ «دوئلِ برابر» (`GAME-DESIGN.md`) می‌گوید دوئلِ آنلاین باید فقط با دانش و
سرعت برنده شود (ضدِ pay-to-win). پس پاورآپ‌ها فعلاً فقط روی **بازیِ تک‌نفره**
(`QUICK`/`BOMB`) اعمال می‌شوند. پاورآپ‌های مخصوصِ دوئل (پرسِ سنگین، دفعِ بمب)
اگر لازم شد باید جداگانه و با بالانسِ دقیق اضافه شوند.

## ۲) پاورآپ‌ها (`powerup.constants.ts`)

| نوع | اثر | هزینه |
|---|---|---|
| `fifty` ✂️ نصف‌نصف | یک گزینهٔ غلط حذف می‌شود | ۱ کارت یا ۳۰ سکه |
| `extra_time` ⏱️ وقت اضافه | ۷ ثانیه به deadline اضافه می‌شود | ۱ کارت یا ۳۰ سکه |
| `swap` 🔄 تعویض سؤال | سؤال با یکی دیگر عوض و تایمر ری‌ست می‌شود | ۲ کارت یا ۶۰ سکه |

## ۳) API

| متد | مسیر | کار |
|---|---|---|
| GET | `/powerups` | کاتالوگ (عمومی، برای UI) |
| POST | `/powerups/rounds/:roundId` | استفاده (پشتِ JWT) — body `{ type, pay: 'card'\|'coin' }` |

پاسخ بر اساس نوع:
- `fifty` → `{ type, removedOptionId }` (کلاینت آن گزینه را محو می‌کند)
- `extra_time` → `{ type, deadlineAt, addedSeconds }`
- `swap` → `{ type, round: { roundId, question, options[], deadlineAt, serverNow } }`
  (گزینه‌ها **بدونِ** isCorrect)

## ۴) قواعدِ اعمال (سرور)

1. کاربرِ احرازشده باید بازیکنِ همان مَچ باشد.
2. راند نباید جواب‌داده‌شده یا منقضی باشد.
3. **هر پاورآپ فقط یک‌بار در هر راند** — با یک set در Redis (`powerup:<roundId>`)
   رزرو می‌شود.
4. هزینه از کیف‌پول کم می‌شود؛ **اگر پرداخت ناموفق بود، رزرو آزاد می‌شود** تا
   تلاشِ مجدد ممکن بماند.
5. `extra_time`/`swap` علاوه بر DB، قفلِ deadline در Redis را هم به‌روز می‌کنند.

## ۵) تستِ دستی (خلاصه)
```bash
curl localhost:3000/powerups            # کاتالوگ
curl -X POST localhost:3000/powerups/rounds/<roundId> \
  -H "Authorization: Bearer <T>" -H 'Content-Type: application/json' \
  -d '{"type":"fifty","pay":"coin"}'
```
رفتارِ تأییدشده: سه اثر، کسرِ هزینه، ردِ تکرارِ هر پاورآپ در یک راند، ردِ موجودیِ
ناکافی (سکه/کارت)، و آزادسازیِ صحیحِ رزرو پس از خطای پرداخت.

## ۶) کارهای بعدی
- UI: `PowerupBar` روی صفحهٔ سؤالِ تک‌نفره.
- پاورآپ‌های بیشتر: `👁️ دید داور` (یک سؤالِ بعد را ببین)، `🧤 دستکش`، و
  پاورآپ‌های دوئلی با بالانسِ جدا.
- منبعِ کارت: استریک/اچیومنت (پیاده شده) + بستهٔ فروشگاه (Poolakey، فاز بعد).
