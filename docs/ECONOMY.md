# فوتبالیکا — اقتصاد بازی (Economy Balancing)

> ⚠️ این اعداد **نقطهٔ شروع** برای بالانس‌اند، نه نهایی. تعادلِ واقعی فقط با
> تستِ کاربر واقعی به‌دست می‌آید. این‌جا هدف، یک پایهٔ منطقیِ قابل‌تنظیم است.

اصلِ راهنما: یک کاربرِ **رایگان** باید حس کند در حال پیشرفت است (نه گیر‌کرده)،
و یک کاربرِ **پرداخت‌کننده** باید سرعت بخرد، نه «تنها راهِ پیشرفت» را.

---

## ۱) ارزها و نرخِ کسب

| ارز | منبع | مقدارِ تقریبی |
|---|---|---|
| 🪙 سکه | بردِ بازی سریع | ۴۰–۶۰ |
| 🪙 سکه | باختِ بازی سریع | ۱۰–۱۵ (تا دلسرد نشود) |
| 🪙 سکه | هر جوابِ درست | ۵ |
| 🪙 سکه | جایزهٔ روزانهٔ استادیوم | ۳۰–۲۰۰ (بسته به سطح) |
| 🎽 هوادار | بردِ دوئل | ۸۰–۱۵۰ |
| 🎽 هوادار | باختِ دوئل | ۱۰–۲۰ |
| ⚡ کارت | استریکِ ۷ روزه | ۱ کارت |
| ⚡ کارت | اچیومنت | ۱–۳ کارت |
| ⚡ کارت | صعودِ دسته | ۵ کارت |

هدفِ طراحی: کاربرِ فعال روزانه حدود **۲۰۰–۳۰۰ سکه** جمع کند.

---

## ۲) جان (❤️) — تنظیم‌کنندهٔ ریتم

- سقفِ پایه: **۵ جان**.
- هر بازی: **۱ جان**.
- بازیابیِ خودکار: **۱ جان هر ۲۰ دقیقه** (با ارتقای زمین تمرین کمتر می‌شود).
- پرکردنِ فوری: ۵۰ سکه، یا تماشای تبلیغ، یا کارت.

این یعنی کاربرِ رایگان در یک نشست حدود ۵ بازی پشت‌سرهم می‌کند، بعد یا صبر
می‌کند یا خرج می‌کند. این «ریتم» بازی را می‌سازد و از فرسودگی جلوگیری می‌کند.

---

## ۳) ارتقای بخش‌های باشگاه

هزینه‌ها **نمایی** بالا می‌روند (پایه × ضریب^سطح). فرمولِ پیشنهادی:

```
هزینهٔ سطحِ بعد = round( baseCost × 1.6^(currentLevel-1) )
```

نمونه برای «زمین تمرین» (baseCost = 100):

| سطح | هزینه (سکه) | اثر: بازیابی جان |
|---|---|---|
| ۱→۲ | ۱۰۰ | هر ۱۸ دقیقه |
| ۲→۳ | ۱۶۰ | هر ۱۶ دقیقه |
| ۳→۴ | ۲۵۶ | هر ۱۴ دقیقه |
| ۴→۵ | ۴۱۰ | هر ۱۲ دقیقه |
| ۵→۶ | ۶۵۵ | هر ۱۰ دقیقه |
| ...→۱۰ | ~۴۳۰۰ | هر ۵ دقیقه (سقف) |

جدولِ اثرِ هر بخش (سطح ۱ تا ۱۰، خطی مگر ذکر شود):

| بخش | baseCost | اثر در سطح ۱ | اثر در سطح ۱۰ |
|---|---|---|---|
| 🏋️ زمین تمرین | ۱۰۰ | جان هر ۲۰ دق | جان هر ۵ دق |
| 🏟️ استادیوم | ۱۵۰ | درآمد روزانه ۳۰ | درآمد روزانه ۲۰۰ + ×۱٫۵ هوادار |
| 🎓 آکادمی جوانان | ۱۲۰ | ۱ راهنمایی رایگان | ۱ راهنمایی + سؤال گرم‌کن آسان‌تر |
| 📊 اتاق آنالیز | ۲۰۰ | دیدِ دستهٔ بعد ۱بار | دیدِ دستهٔ بعد ۳بار |
| ⚕️ پزشکیار | ۱۳۰ | جریمهٔ باخت −۱۰٪ | جریمهٔ باخت −۵۰٪ |
| 📡 دپارتمان رسانه | ۱۱۰ | +۱۰٪ سکهٔ تبلیغ | +۵۰٪ سکهٔ تبلیغ |

## ۴) مسیرِ صعود در جدول

صعود با «هوادار» و بردِ مسابقاتِ آن دسته:

| دسته | هوادارِ لازم برای صعود | سختیِ سؤال | ضریبِ جایزه |
|---|---|---|---|
| دستهٔ سه | ۱٬۰۰۰ | آسان‌غالب | ×۱٫۰ |
| دستهٔ دو | ۳٬۰۰۰ | متوسط | ×۱٫۳ |
| دستهٔ یک | ۸٬۰۰۰ | متوسط‌سخت | ×۱٫۶ |
| لیگ برتر | ۲۰٬۰۰۰ | سخت | ×۲٫۰ |
| لیگ قهرمانان | — (قله) | خیلی سخت | ×۳٫۰ |

سقوط: اگر در لیگ هفتگی امتیازِ کافی نگیری، یک دسته پایین می‌آیی — این ریسک
هیجانِ فصل را می‌سازد. (اختیاری؛ می‌شود در نسخهٔ اول ملایم‌تر بود.)

## ۵) قیمتِ سوپرپاورها (کارت)

| کارت | هزینه | کمیابی |
|---|---|---|
| ✂️ نصف‌نصف | ۱ کارت یا ۳۰ سکه | رایج |
| ⏱️ وقت اضافه | ۱ کارت یا ۳۰ سکه | رایج |
| 🔄 تعویض سؤال | ۲ کارت یا ۶۰ سکه | متوسط |
| 👁️ دید داور | ۲ کارت یا ۶۰ سکه | متوسط |
| 💣 دفعِ بمب | ۳ کارت | کمیاب |
| 🧤 دستکشِ طلایی | ۴ کارت | کمیاب |

نکته: کارت‌های کمیاب را با سکه نفروش — فقط با کارت. این‌طور «کارت» ارزشش
را حفظ می‌کند و به برد در دوئل معنا می‌دهد.

## ۶) مدلِ درآمد (پیشنهادی، Poolakey)

| محصول | چه می‌دهد |
|---|---|
| بستهٔ سکه | ۵۰۰ تا ۱۰٬۰۰۰ سکه در چند سطحِ قیمتی |
| بستهٔ کارت | مجموعهٔ سوپرپاور |
| حذفِ تبلیغ | تجربهٔ بدون تبلیغ (یک‌بار خرید) |
| بلیتِ فصلی (Battle Pass) | جایزهٔ فصلیِ پلکانی؛ بهترین ارزش برای درآمدِ پایدار |
| باشگاهِ ویژه | اسکین/لوگوهای خاص (کاسمتیک، نه قدرت) |

اصلِ اخلاقی/بقای بازی: **pay-to-skip، نه pay-to-win**. رایگان‌باز باید بتواند
به لیگ قهرمانان برسد؛ پرداخت فقط سریع‌ترش کند. اگر برنده‌شدن را بفروشی،
رقابت بی‌معنا و بازی می‌میرد.

## ۷) پارامترهای قابل‌تنظیم (برای کد)

این‌ها را در یک فایلِ config بگذار تا بدونِ دیپلوی مجدد بالانس کنی:

```ts
export const ECONOMY = {
  lives: { max: 5, regenMinutes: 20, refillCost: 50 },
  coins: { winQuick: 50, loseQuick: 12, perCorrect: 5 },
  fans:  { winDuel: 120, loseDuel: 15 },
  upgrade: { costMultiplier: 1.6 },
  promotion: { fansNeeded: [1000, 3000, 8000, 20000] },
  rewardMultiplier: [1.0, 1.3, 1.6, 2.0, 3.0], // بر اساس دسته
};
```
