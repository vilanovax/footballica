# فوتبالیکا — پنل ادمین (Next.js)

CMS سؤال‌ها برای فاز ۱: فهرست/ساخت/تأیید/حذفِ سؤال، آمارِ نرخِ پاسخِ درست، داشبورد.
RTL و فارسی، فونتِ لوکالِ وزیرمتن.

## استک
Next.js 14 (App Router) + TypeScript سخت‌گیرانه. بدون کتابخانهٔ UI (CSS سبک).

## راه‌اندازی

```bash
cd admin
cp .env.example .env.local     # NEXT_PUBLIC_API_BASE_URL را تنظیم کن
npm install
npm run dev                    # http://localhost:3001
```

اول **بک‌اند** (`../backend`) را بالا بیاور. هنگام ورود، **کلیدِ ادمین** را وارد کن
(همان `ADMIN_KEY` بک‌اند). کلید فقط در `localStorage`ِ مرورگر نگه‌داری می‌شود و در
هر درخواست به‌صورت هدرِ `x-admin-key` فرستاده می‌شود.

## صفحه‌ها

| مسیر | کار |
|---|---|
| `/login` | ورود با کلیدِ ادمین |
| `/` | داشبورد (شمارشِ سؤال/دسته/پاسخ) |
| `/questions` | فهرست + تأیید/لغو تأیید/حذف + فیلتر + آمار |
| `/questions/new` | ساخت سؤال (اعتبارسنجی «دقیقاً یک گزینهٔ درست») |

## اتصال به بک‌اند
از endpointهای `/admin/*` استفاده می‌کند (همه پشتِ `AdminGuard`). قرارداد در
`lib/types.ts` و کلاینت در `lib/api.ts`.
